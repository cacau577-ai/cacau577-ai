# 👨‍💻 Caua Almeida


**`Desenvolvedor FullStack`**

Me chamo Cauã Almeida, sou natural de Salvador, Bahia. Concluí o curso técnico em TI pelo SENAI e, atualmente, estou cursando Bacharelado em Ciência da Computação na Unijorge.

Sou apaixonado por tecnologia e empreendedorismo. Atualmente, lidero minha própria equipe no desenvolvimento de soluções digitais, como sites institucionais, e-commerces ex: [Pathy Glamour] (https://gsreisz.github.io/Pathy-Glamour-Store/), cardápios e agendas digitais. Além do desenvolvimento de software com foco em Python, Java e SQL, possuo sólida experiência em infraestrutura e manutenção de hardware, atuando desde a montagem até o diagnóstico avançado de sistemas.
